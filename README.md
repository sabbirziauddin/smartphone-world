# smartphone-world

git clone <https://github.com/sabbirziauddin/smartphone-world>

install dependencies
 npm install

 Build
 npm run build
 Start
 npm run start
 start (development)
 npm run start:dev
 Linting
 npm run lint
npm run lint:fix
Formatting
npm run prettier
npm run prettier:fix
environment variable
PORT = Port number
DB_URL = Database url
